ANTIQUIFY.js

Antiquify is a jQuery plugin developed by Matthew Gonzalez that takes text and modifies the styles to make it look like it came straight off an old press. Randomized aberrations help create the illusion that the type was laid by hand using antique printing techniques. Special thanks to the fellas at Paravel and their Lettering.js, on which this plugin is heavily based. NOTE: this is currently a work in progress. Stay tuned.

WHY? Because I like old books and the tactile quality found therein. Imperfections tend to humanize these sorts of things. A simple example of an old book can be found here.
